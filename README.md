# UYeong

UYeong은 블로그 기능을 가진 웹 애플리케이션으로 프론트엔드 개발자를 준비하는 과정에서 제 이야기를 기록하고 사용자와 함께 소통이 가능한 공간을 직접 만들고 싶어 제작하게 되었습니다.
